Questions

  1. What does the server in the client server architecture do?
    The server stores and serves files for the clients.

  2. What does the client do?
    The client requests, recieves and processes data from the server.

  3. What is an IP address?
    An IP address is a unique identifying address used to find devices on the Internet.

  4. What is DNS?
    DNS stands for Domain Name Server and acts like a Yellow Pages for the internet, storing domain names together with their related IP address.

  5. What is the difference between the Internet and WWW?
    The Internet is the physical network connecting computers all over the world. WWW is an application of the Internet.

  6. Explain what HTML is.
    HTML (HyperText Markup Language) is the language used to write the structure and content of websites.
